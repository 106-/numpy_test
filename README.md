# numpy_test
2018年度の八峰祭で使うスライドのプログラム C++ | python | python+numpy での計算速度比較


C++のプログラムがガバガバコードなのは仕様

# 実行結果例
```
$ ./run-all.sh
===== C++ =====
real 16.23
user 16.22
sys 0.00
===== Python + numpy =====
real 1.99
user 2.06
sys 0.35
===== Python =====
real 413.10
user 412.99
sys 0.04
```
